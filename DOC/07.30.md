### IIFE 패턴

즉시 호출되는 함수 표현식 (이름으로 호출과정 생략).
상위 영역과 별도로 구분되는 지역을 만들어 줌. (외부와 차단)

```js
// IIFE
(function(global, _){
	// 전역과 구분되는 지역

	var body = _.$('body'),
		app = _.$('#app');

	_.css(app, 'background: red; color: white; height: 100vh;');

})(window, window.yamoo9);
```
